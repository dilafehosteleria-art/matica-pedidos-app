# Matica Pedidos App

Proyecto limpio en Next.js conectado a Supabase.

## Variables de entorno

Usa estas en Vercel:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## Arranque local

```bash
npm install
cp .env.example .env.local
npm run dev
```

La tabla de Supabase usada por esta versión es `Pedidos`.
